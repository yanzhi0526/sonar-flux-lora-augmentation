# Limitations

- The original real side-scan sonar images are not redistributed due to data-licensing restrictions.
- Generated samples are synthetic and may not reflect all real-world sonar conditions.
- Seed-level prompt metadata may be incomplete if not available in the exported generation logs.
- FLUX model weights and LoRA weights are not released in the current version.
- Exact reproduction of real-only validation/test detection metrics requires access to the original real dataset.
- The repository result tables are manuscript summaries; users should rerun training if adapting the workflow to a different real sonar dataset.
